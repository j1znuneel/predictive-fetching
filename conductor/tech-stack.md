# Technology Stack: PrefetchAI

## Frontend
- **React 18**: The core UI framework, used for building modular and reactive predictive components.
- **JavaScript (ESM)**: Utilizes modern JavaScript features for performant, dependency-free modules.

## Tooling & Infrastructure
- **Vite 5**: High-performance build tool and development server for rapid iteration.
- **Vitest 4**: Vite-native testing framework for unit and integration tests.
- **React Testing Library**: Used for verifying hook and component behavior in a simulated DOM environment.

## Prediction Logic
- **Kinematic Engine**: Custom vanilla JavaScript logic for real-time mouse tracking using `requestAnimationFrame`.
- **Markov Tracker**: Lightweight, `localStorage`-persistent Markov Chain implementation for behavioral sequencing.
